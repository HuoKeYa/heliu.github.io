    华为OBS上传组件

    一、快速了解¶

    功能特性：¶

    • 可视化连接华为云OBS对象存储；
    • 支持直传到华为云OBS对象存储；
    • 支持批量上传文件；
    • 支持手机端拍照上传、选择文件上传；
    • 支持指定存储桶；
    • 支持指定文件上传目录；
    • 支持指定上传的文件类型；
    • 支持文件预览、文件下载；
    • 支持快速拼接Form表单；
    • 提供不同格式的文件地址，方便数据处理；

    • 组件发布前经过了单元测试、集成测试、代码评审，组件质量更有保障。

    • OBS上传组件，旨在提供图形化配置第三方对象存储的能力，开发者仅需填写4个配置参数，即可连接华为云对象存储服务（简称OBS），OBS上传组件集成了文件上传、预览、下载等业务功能，简化开发，降低开发者学习成本。

    • 组件一般不是独立的存在，它可以与Form表单进行拼接、连接，适应大部分业务场景。

    二、快速上手¶

    两个关键步骤：配置连接OBS，使用组件实现业务功能

    2.1 配置：连接OBS¶

    前提：已购买OBS服务，购买链接：https://www.huaweicloud.com/product/obs.html

    Huawei OBS 的帮助文档非常详细：https://support.huaweicloud.com/qs-obs/obs_qs_0003.html

    在上述文档中，我们只要获取红框中这四个信息，填写在【配置页】中就可以了！

    点击【配置页】的「保存」按钮，组件会检查配置参数是否有效。

    如果有提示“连接可用！”就说明配置成功了！如配置错误会有弹窗提示。

    配置成功后可以点击「预览页面」使用组件上传图片，上传后可在OBS Brower+（OBS的图形化管理工具）中查看刚上传的图片。

    2.2 使用：实现业务功能¶

    前提：掌握无远必修教程

    完成连接我们已经可以上传图片，并且上传图片成功后，我们在**组件变量（STRURL、ARRURL）**中可以获取到OBS返回的文件url。我们可以使用组件变量来实现业务功能。

    STRURL是字符串，多个文件url以逗号分隔；ARRURL是数组，一个数组元素存入一个文件url。

    🔑 Create¶

    当我们上传文件后，希望结合业务场景将文件url与其他信息一同存取至数据库中，可以使用变量STRURL。

    下图提供INSERT示例语句

    INSERT INTO mat(
    code,
    name,
    img_url
) VALUES
(
    @12-CODE@,
    @12-NAME@,
    @13-STRURL@ -- 将“11”修改为您使用OBS组件的窗口号
);

    🔑 Read¶

    新增数据后，我们希望查看这些文件，可以在【配置页】中配置数据源，读取已存入数据库的文件url信息，通过组件加载对应文件。

    特别注意使用下图提供的SQL语句，才可以方便地将文件URL转换为组件要求的数据格式，下述只需要把 mat 数据表替换为你的真实数据表，mat.img_url 替换为你的真实文件url字段，where条件替换为你的真实id字段即可：

    SELECT
    substring_index( substring_index( mat.img_url, ',', t_index.help_topic_id + 1 ), ',',- 1 ) AS source -- 注意必须as为source才能识别
FROM
    mat
    JOIN mysql.help_topic t_index ON t_index.help_topic_id < ( length( mat.img_url ) - length( REPLACE ( mat.img_url, ',', '' ))+ 1 ) 
WHERE
    mat.`code` = @14-CODE@ --这个变量是用户点击其他组件的某行记录即可传递过来的id，使用此id找到文件url

    输入语句按上图点击两处保存（「保存数据源」和「保存」）后，就可以预览查看效果了！

    🔑 Update¶

    当我们查看数据后，如果希望修改文件，并保存至数据库，原理及流程与Create相似，重新上传文件后，**使用变量STRURL**更新。

    🔑 Delete¶

    本组件不提供删除OBS文件的功能，因为OBS文件未经备份直接删除，文件不可恢复，所以一般建议使用OBS Brower+（OBS的图形化管理工具）进行删除操作。

    三、扩展应用（配置页参数说明）¶

    3.1 文件夹📁¶

    对象存储本无文件夹的概念，为照顾用户的使用习惯，使用前缀作为虚拟文件夹（即：文件夹名/文件名），并不需要提前建立文件夹。

    OBS上传组件经过简单的配置后，可以上传至对应的文件夹，以此实现分开存储管理不同业务文件的功能。

    例如：有两个功能页面使用了此组件，A页面的OBS上传组件可以设置上传至A文件夹，B页面的可以设置上传至B文件夹。

    3.2 立即上传⏫¶

    勾选即代表在本地选取文件后，立即上传至OBS。

    3.3 文件类型🗂¶

    OBS上传组件可以指定用户上传的文件类型，特别注意**需要输入mime类型**，而非文件扩展名。多个文件类型使用逗号分隔。

    常见的mime类型可以查看这份文档：常见 Mime Type 参考表

    如果希望指定所有的图片类型，可以用：“image/*”，当使用该类型时，移动端可以选择使用“拍照”📷 或“相册” 💾 。

    3.4 表单拼接📋¶

    OBS上传组件通常需要与表单其他字段拼接在一起，这个时候就需要使用【表单拼接】功能来满足此业务场景。
